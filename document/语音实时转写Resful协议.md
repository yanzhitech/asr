# 实时转写Restful接口协议

## 1. 客户端：

```
{
    "appid":string,
    "userid":string,
    "ts":string,
    "sid":string, //每段连续音频，一个唯一值
    "data":binary message //最后发空字段结束
}
```

## 2. 服务端：

```
{
        "action":string,
        "code":string,
        "data":json,
        "desc":string,
        "sid":string
}
```
data 样例：


| 字段 | 含义 | 描述 |
| ------ | ------ | ------ |
| bg | 句子开始时间 | 中间结果的bg为准确值 |
| ed | 句子结束时间 | 中间结果的ed为0 |
| w | 词识别结果 |  |
| wp | 词标识 | n-普通词；s-顺滑词；p-标点 |
| wb | 词开始时间 | 中间结果的 wb 为 0 |
| we | 词结束时间 | 中间结果的 we 为 0 |
| type | 结果类型标识 | 0-最终结果；1-中间结果 |
| seg_id | 转写结果序号 | 从0开始 |
